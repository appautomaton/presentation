# Founder — Modern Blue

Clean, friendly, modern. For startup pitches, board updates, product narratives, growth stories. One blue hue with cyan accent. Light canvas.

## Core Palette

| Swatch | Hex | Token | L | Use |
|---|---|---|---|---|
| | `#0B2D72` | Navy | 25% | Headlines, primary text, emphasis |
| | `#0992C2` | Accent | 45% | Taglines, metrics, icon highlights, logo elements |
| | `#D4DEE8` | Structure | 85% | Dividers, card borders |
| | `#F5F8FB` | Canvas | 98% | Default slide background |

Text neutrals are true grays.

| Swatch | Hex | Token | L | Use |
|---|---|---|---|---|
| | `#333333` | Text | 20% | Body copy on light surfaces |
| | `#666666` | Gray | 40% | Detail text, captions |

## Signal Colors

For KPI cards, metrics, and data visualization.

| Swatch | Hex | Token |
|---|---|---|
| | `#0992C2` | Blue — primary data series (= Accent) |
| | `#2AADAD` | Teal — secondary data series |
| | `#6651B8` | Violet — third data series |

For status indicators (scorecard cards, health checks):

| Swatch | Hex | Token |
|---|---|---|
| | `#2E9E5A` | Green — positive, on-track |
| | `#E8A317` | Amber — caution, mixed |
| | `#D04545` | Red — risk, off-track |

## Text Pairing

- On Canvas: Navy headlines, Text body.
- Accent for taglines, metrics, and icon highlights.
- Gray for detail text and captions only.
- Do not set body text in Accent.

## Typography

| Font | Role | Use | Avoid |
|---|---|---|---|
| Arial | Display | Headlines, section titles, metrics | — |
| Arial | Body | Body copy, bullets, tables, source lines | — |
