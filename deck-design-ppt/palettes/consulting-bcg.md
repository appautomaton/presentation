# BCG — Green-Anchored

Process-transparent, framework-first. Green rule below every action title (signature). Panel mini-headers with local green underline on multi-panel slides. "Trust us because you can see exactly how we got here."

## Core Palette

Two greens: structural dark green for authority, rule green for visual rhythm. Everything else is gray or white.

| Hex | Token | Use |
|---|---|---|
| `#1A1A1A` | Midnight | Section divider overlay background |
| `#0F6B4F` | Structure | Section labels (uppercase), structural accent, panel headers |
| `#6AB648` | Rule | Header rule, panel mini-rules — the BCG signature |
| `#D0D0D0` | Border | Panel borders, table gridlines |
| `#F5F6F4` | Warm gray | Alternate row fills, low-salience zones |
| `#FFFFFF` | White | Default slide canvas |

## Text

| Hex | Token | L | Use |
|---|---|---|---|
| `#111111` | Text | 7% | Action titles, body text |
| `#5F6368` | Gray | 40% | Subtitles, notes, axis labels |

## Text Pairing

- On White: Text headlines and body. Section labels in Structure, uppercase, letter-spaced.
- On Midnight: `#FFFFFF` first, Warm gray second.
- Gray for subtitles, notes, and axis labels only.
- Do not set body text in Rule green.

## Charts

Teal-green (`#1F7A5A`) for primary series. Gray for secondary. BCG charts are flat 2D, no gradients, no shadows. Max 3 series colors.

## Evaluation Matrices

The scorecard matrix is BCG's signature exhibit (~20% of analytic slides). Cell fills use RAG washes at low opacity — not solid fills:

| Hex (wash) | Rating |
|---|---|
| `rgba(62,158,90,0.18)` | Positive / favorable |
| `rgba(231,179,61,0.20)` | Caution / mixed |
| `rgba(200,90,84,0.16)` | Negative / constraint |

Thin gray gridlines (~0.5pt). Bold row/column headers. The color pattern across the grid IS the visualization.

## Typography

Arial for everything — display and body. No second font.

## Mode Variations

| Mode | What changes |
|---|---|
| **Classic Program Book** | Full green-on-white. Roman numeral sections. Dark green dividers. L2–L3. |
| **Modern Impact** | Photo-based section dividers with Midnight overlay. Section numbers 60–80pt. L1–L2. |
| **Working Binder** | Yellow caveat banners for methodology limits. "DRAFT FOR DISCUSSION" tag. L2–L3. |

Colors and typography are the same across all modes. Modern mode adds photography on dividers.
