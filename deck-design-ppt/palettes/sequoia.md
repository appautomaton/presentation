# Sequoia — Dark + Red

Dark, dramatic, high-stakes. For crisis messaging, downturn memos, strategic inflection points, "burn/runway" rhetoric. Red accent on near-black canvas. Georgia serif for authority.

## Core Palette

Dark canvas with warm undertone. No pure black, no pure white.

| Swatch | Hex | Token | L | Use |
|---|---|---|---|---|
| | `#0D0B0A` | Midnight | 4% | Default slide background |
| | `#1A1614` | Surface | 9% | Card backgrounds, elevated surfaces |
| | `#2A2420` | Border | 14% | Card borders, dividers, accent bar backgrounds |
| | `#E8554D` | Accent | 55% | Headlines, accent bars, focal metrics |

Text on dark surfaces.

| Swatch | Hex | Token | L | Use |
|---|---|---|---|---|
| | `#F5F0E5` | Light | 95% | Primary body text, card labels |
| | `#B0B0B0` | Gray | 69% | Secondary text, context lines |
| | `#999999` | Fine | 60% | Captions, footnotes |

## Signal Colors

Red IS the signal. Everything else is gray.

| Swatch | Hex | Token |
|---|---|---|
| | `#E8554D` | Red — primary emphasis (= Accent) |
| | `#D4554D` | Dark red — subtitle emphasis, secondary accent |
| | `#888888` | Muted — non-focus, context |

## Text Pairing

- On Midnight: Light for body, Accent for headlines and metrics.
- Gray for secondary text and context lines.
- Fine for captions and footnotes only.
- Do not set extended body text in Accent (hard to read at small sizes on dark).

## Typography

| Font | Role | Use | Avoid |
|---|---|---|---|
| Georgia | Display | Headlines, title text (serif, dramatic) | Dense body text, tables |
| Arial | Body | Body copy, bullets, detail text, source lines | Hero titles |

Georgia is ~10–15% wider per character than Arial. Reduce character budgets accordingly.
