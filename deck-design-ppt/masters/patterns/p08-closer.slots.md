# P08 — Closer — Slot Budgets

## Consulting

- **For:** recommendation + next step.
- **Slots (budgets):**
  - Hero line: <= 90 chars.
  - Supporting line(s): <= 120 chars total.
  - Contact block: 2–4 short lines (<= 45 chars each).

## Founder

- **For:** fundraising ask + contact.
- **Slots (budgets):**
  - Ask line: <= 60 chars.
  - Proof line: <= 120 chars.
  - Contact: <= 4 lines.

## Sequoia

- **For:** final imperative line (high stakes).
- **Slots (budgets):**
  - Hero: <= 40 chars.
  - Supporting: <= 120 chars.

**Data schema:**
```js
{ title, body, nextSteps?: [string] }
```
