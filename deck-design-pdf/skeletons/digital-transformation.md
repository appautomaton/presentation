# Digital / AI Transformation — Skeleton

## Governing Thought

**Affirmative:**
```
"[Company] should invest $[X]M over [timeframe] in [N] digital capabilities,
 prioritized by impact × feasibility, to capture $[Y]M in annual value —
 starting with [top 2-3 use cases] in [Phase 1 timeframe]."
```

**Negative:**
```
"[Company] should not pursue enterprise transformation because [foundational blocker]
 makes large-scale investment premature. Instead, invest $[X]M in [prerequisite]."
```

## Pillar Architecture

| # | Pillar | Sub-Question | Transition Out |
|---|--------|-------------|----------------|
| 1 | **Digital Maturity Diagnostic** | Where do we stand vs. peers and leaders? | Pivot → "gap is $[X]M value at stake" |
| 2 | **Use Case Prioritization** | Which of 40-60 use cases should we fund? | Narrowing → "of N, M pass threshold" |
| 3 | **Architecture & Capability** | What tech, talent, operating model? | Escalation → "value confirmed, execution is the constraint" |
| 4 | **Transformation Roadmap** | How to sequence, fund, govern? | Closes to recommendation |

**Core device: AI Impact-Feasibility Matrix** — Every use case plotted on impact (y) × feasibility (x). Upper-right = quick wins (Phase 1). Upper-left = strategic bets (Phase 2-3).

**Digital Value Chain (6 stages):** Data Capture → Integration → Analytics/AI → Decision Automation → Digital Channels → Feedback/Learning. Score each stage; find "broken links" (2+ point drops).

## Kill Conditions

| # | Trigger | Pivot To |
|---|---------|---------|
| KC-1 | Data infrastructure absent (value chain stages 1-2 score <2.0) | Foundation-first strategy |
| KC-2 | Required talent >3× available pool, hiring blocked | Outsource-led or acqui-hire |
| KC-3 | All scenarios below hurdle rate IRR | Targeted 2-3 defensive use cases only |
| KC-4 | >60% of BU leaders actively resist, sponsor lacks authority | Organizational redesign before technology |
| KC-5 | 1-2 competitors at >4.5/5.0 maturity with network effects | Differentiate on non-digital axis or acquire |

## Quality Gates

- [ ] Value chain broken link identified with quantified downstream impact
- [ ] Impact-Feasibility Matrix has ≥25 scored use cases with clear quadrant separation
- [ ] Every dollar traces to a use case; every value estimate decomposes into rev/cost/risk
- [ ] Phase 1 generates measurable value within 6 months (self-funding test)
- [ ] Talent plan within 15% of available pool per skill
- [ ] Operating model slide present; adoption KPIs in scorecard
- [ ] Benchmark includes ≥2 digital-native competitors
- [ ] Architecture evaluates ≥2 vendors per layer (no single-vendor bias)
