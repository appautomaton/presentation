# Profitability / Cost Transformation — Skeleton

## Governing Thought

**Affirmative:**
```
"[Company] can improve EBITDA margin by [X]bps through [N] cost levers
 across [scope], capturing $[Y]M in annualized savings within [timeframe],
 because [root cause] creates addressable structural opportunity."
```

**Negative:**
```
"[Company]'s margin gap to peers is [X]bps, but only [Y]bps is addressable
 without [structural barrier]. The priority is [alternative] before cost optimization."
```

## Pillar Architecture

| # | Pillar | Sub-Question | Transition Out |
|---|--------|-------------|----------------|
| 1 | **Benchmark the Gap** | How does our cost structure compare to peers? | Pivot → "gap is real, here's the prize" |
| 2 | **Explain the Gap** | What root causes drive the cost disparity? | (flows directly) |
| 3 | **Size the Prize** | Where are the largest addressable opportunities? | Narrowing → "of N levers, M survive" |
| 4 | **Design Initiatives** | Which initiatives, in what sequence? | Escalation → "can we execute?" |
| 5 | **Prove the P&L Math** | What is the phased financial impact? | Closes to recommendation |

## Kill Conditions

| # | Trigger | Pivot To |
|---|---------|---------|
| KC-1 | Gap <200bps after normalization | Cost optimization only (narrow scope, fewer slides) |
| KC-2 | >60% of gap in non-addressable categories (regulatory, structural) | Revenue growth strategy instead of cost |
| KC-3 | Cost reduction requires headcount >15% of workforce | Restructuring engagement (different skeleton) |
| KC-4 | Peer benchmark set is invalid (no true comparables) | First-principles cost modeling instead of benchmarking |

## Quality Gates

- [ ] Peer benchmark uses ≥5 comparable companies with transparent selection criteria
- [ ] Cost decomposition reaches actionable granularity (cost center, not just category)
- [ ] Every initiative has: owner, investment, savings, timeline, confidence level
- [ ] P&L bridge sums correctly from initiative-level to total
- [ ] Downside scenario still clears the minimum acceptable margin target
- [ ] Waterfall bridges (≥3 expected) are internally consistent
- [ ] Helicopter test: titles of slides 2, 5, 10, 16, 19 tell the complete story
