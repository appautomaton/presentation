# M&A Strategy

How to evaluate, execute, and integrate acquisitions. Covers the decision to acquire, target selection, synergy quantification, and post-close integration.

---

## 1. Buy vs. Build vs. Partner

```
Need capability or market X
├── Can we build it within the required timeframe?
│   ├── Yes, and we have the talent → BUILD
│   └── No, or talent gap too wide → continue
├── Is ongoing access sufficient, or do we need ownership?
│   ├── Access sufficient → PARTNER (JV, license, alliance)
│   └── Need ownership → ACQUIRE
└── Would a competitor acquiring the target harm us?
    ├── Yes → urgency increases, lean ACQUIRE
    └── No → evaluate on merit
```

| Criterion | Weight | Build | Partner | Acquire |
|---|---|---|---|---|
| Speed to market | 20% | Score 1-5 | Score 1-5 | Score 1-5 |
| Total cost (5-year NPV) | 20% | | | |
| Strategic control | 15% | | | |
| Risk level | 15% | | | |
| Talent / IP access | 10% | | | |
| Revenue synergy potential | 10% | | | |
| Reversibility | 10% | | | |

**Build wins** when time-to-market is >18 months and acceptable, the capability is core to long-term advantage, and internal talent exists or is acquirable within 12 months.

**Partner wins** when speed matters but ownership doesn't, the capability is non-core or rapidly evolving, or the partner brings market access that would take years to replicate.

**Acquire wins** when time-to-market is critical, the capability is difficult to replicate, a competitor acquiring the target would damage your position, or integration complexity is manageable.

## 2. Deal Thesis

Every acquisition needs a one-paragraph thesis that answers three questions: Why this target? Why now? Why us?

### Thesis archetypes

| Archetype | Logic | Key risk |
|---|---|---|
| Scale | Combine to reduce unit costs and increase pricing power | Dis-synergies in culture or operations |
| Scope | Add adjacent capability, product, or market | Integration complexity destroys the adjacency value |
| Capability | Acquire talent, technology, or IP that can't be built fast enough | Key people leave post-close |
| Consolidation | Roll up fragmented market for scale economics | Overpayment in competitive auction |
| Transformational | Fundamental business model shift | Execution overwhelms the organization |

The thesis determines what matters in diligence. A scale deal lives or dies on cost synergies. A capability deal lives or dies on talent retention. Diligence that doesn't test the thesis is wasted effort.

## 3. Target Screening

### Screening funnel

1. **Long list** (50-100): sector scan filtered by size, geography, and strategic fit.
2. **Medium list** (10-20): apply must-have criteria. Binary pass/fail.
3. **Short list** (3-5): score on weighted attractiveness criteria. Rank.

### Must-have criteria (binary)

- Minimum revenue or capability threshold
- Geographic or regulatory fit
- No known deal-breakers (litigation, sanctions, structural issues)
- Owner willing to sell within the relevant timeframe

### Attractiveness scoring (weighted)

| Criterion | Weight | Target A | Target B | Target C |
|---|---|---|---|---|
| Strategic fit | 25% | | | |
| Financial profile (growth, margin, returns) | 20% | | | |
| Synergy potential | 20% | | | |
| Integration complexity | 15% | | | |
| Valuation / affordability | 10% | | | |
| Cultural compatibility | 10% | | | |

## 4. Synergy Identification and Quantification

### Synergy types

**Revenue synergies** (harder to capture, longer to realize):
- Cross-sell existing products to acquired customer base
- Access new geographies or channels
- Combined offering that neither could sell alone
- Pricing power from reduced competition

**Cost synergies** (more certain, faster to realize):
- Eliminate duplicate functions (finance, HR, IT, facilities)
- Procurement savings from combined purchasing volume
- Manufacturing or distribution consolidation
- Technology platform rationalization

### Quantification discipline

- Estimate each synergy as a range, not a point: "$8-12M in procurement savings."
- State the assumptions that drive the range and which ones, if wrong, change the number materially.
- Apply a realization haircut: historical base rate is 60-80% of identified synergies actually captured.
- Timeline matters: most cost synergies take 12-24 months; revenue synergies take 24-36 months. Model the ramp.
- One-time costs to achieve (restructuring, systems integration, retention packages) offset Year 1 savings. Net these out.

### Integration mode selection

The degree of integration depends on two factors: strategic interdependence (how much the acquirer and target need to share to create value) and organizational autonomy (how much the target's culture and operating model are worth preserving).

| Mode | When | Risk |
|---|---|---|
| **Absorption** (full merge) | High interdependence, low autonomy value | Destroys target culture; talent flight |
| **Preservation** (standalone) | Low interdependence, high autonomy value | Synergies limited to financial; no operational integration |
| **Symbiosis** (selective integration) | High interdependence AND high autonomy value | Most complex; requires granular decisions on what to integrate vs preserve |

Default to symbiosis when unsure — it forces explicit choices per function. Absorption and preservation are edge cases that sound simpler but often fail by ignoring the dimension they de-prioritize.

## 5. Valuation Cross-Check

Apply the triangulation rule: use at least two valuation methods (DCF, comps, precedent transactions, SOTP, LBO) and explain any divergence. Present a valuation range, not a single number. See [financial-analysis.md](financial-analysis.md) for discount rate calibration and decision rules.

## 6. Integration Planning

Integration planning starts during diligence, not after close. The first 100 days determine whether the deal thesis survives contact with reality.

### Integration complexity assessment

| Dimension | Low | Medium | High |
|---|---|---|---|
| Systems overlap | Separate, no migration needed | Partial overlap, phased migration | Full platform consolidation |
| Org structure | Bolt-on, minimal change | Reporting lines change for one side | Full reorganization |
| Cultural distance | Similar industry, values, pace | Different operating style | Fundamentally different cultures |
| Customer overlap | Distinct customer bases | Some shared accounts | Significant overlap requiring deconfliction |
| Geographic overlap | Different regions | Partial overlap | Same markets, same offices |

### Day 1 readiness checklist

- Communication plan for employees, customers, vendors
- Interim operating model with clear reporting lines
- Key talent retention packages signed
- Regulatory approvals obtained
- Systems access and business continuity confirmed
- Customer-facing teams briefed on messaging

### 100-day plan structure

- **Days 1-30**: Stabilize. Retain key people, reassure customers, establish integration governance.
- **Days 31-60**: Align. Map processes, identify quick wins, begin synergy capture workstreams.
- **Days 61-100**: Accelerate. Execute first integration milestones, report early synergy results, adjust plan.

### Integration governance

- **Integration Management Office (IMO)**: Dedicated team with full-time lead, clear authority, cross-functional workstream owners.
- **Cadence**: Weekly workstream reviews, biweekly steering committee, monthly board updates.
- **Synergy tracking**: Named owner per synergy line, baseline-to-actual tracking, exception reporting on misses.

## 7. Post-Merger Performance Tracking

| Metric | Baseline | 6-month | 12-month | 24-month |
|---|---|---|---|---|
| Revenue synergies captured | $0 | $ | $ | $ |
| Cost synergies captured | $0 | $ | $ | $ |
| Key talent retained (%) | 100% | % | % | % |
| Customer retention (%) | baseline | % | % | % |
| Integration milestones on track | — | % | % | % |
| Employee engagement score | baseline | score | score | score |

Track against the deal thesis. If the thesis was "capability acquisition," talent retention is the lead indicator. If the thesis was "scale consolidation," cost synergy capture rate is the lead indicator. A deal that hits its financial targets but loses the people or customers it was built on has failed.
